img - папка с изображениями
thumb - иконки изображений, сформированные из папки img с помощью resize.php

index.php - файл построения галереи изображений, обновление количества просмотров формируется через js данные берутся из БД через запрос к server.php

function.php - собраны все использующиеся функции

admin.php - позволяет синхронизировать изображения в папках thumb и img, так же проверить все ли ссылки на изображения есть в БД лишние автоматически удалятся, пропущенные добавятся

phpgb.sql.gz - дамп базы sql
